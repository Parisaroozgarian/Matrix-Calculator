Matrix Calculator Documentation
Matrix Input Format

Enter your matrices using nested square brackets with numbers separated by commas:

For a 2×2 matrix: [[1, 2], [3, 4]]
For a 3×3 matrix: [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
For a 2×3 matrix: [[1, 2, 3], [4, 5, 6]]
Matrix Operations Guide
Basic Operations
  • Addition (+) - Combines two matrices of the same size
  • Subtraction (-) - Subtracts corresponding elements
  • Element-wise Multiplication (*) - Multiplies corresponding elements
  • Matrix Multiplication (@) - Standard matrix multiplication
Single Matrix Operations
  • Transpose - Flips matrix over its diagonal
  • Scalar Multiplication - Multiplies by a number
  • Determinant - Calculates determinant
  • Inverse - Finds inverse matrix
Matrix Size Requirements
  • Addition, Subtraction, Element-wise Multiplication

    Both matrices must have exactly the same dimensions

  • Matrix Multiplication (A @ B)

    Number of columns in A must equal number of rows in B

  • Determinant, Inverse, Eigenvalues, Eigenvectors

    Matrix must be square (same number of rows and columns)
